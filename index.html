Enumerate candidate formulas from observed m/z, explicit charge, tolerance, element bounds, and isotope restrictions.

Search inputs

Tolerance ppm

ppm error = (predicted_mz - observed_mz) / observed_mz * 1,000,000

A formula is accepted if the absolute ppm error is within the selected tolerance.

Formula search space

Use one row per isotope-specific species. Repeated elements are allowed when different isotopes are selected.
